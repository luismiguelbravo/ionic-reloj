REMOVER PLUGIN DE CORDOVA DEL PROYECTO


First, you should list your plugins:

  $ ionic cordova plugin list
With this result, you can simply do:

  $ cordova plugin remove <PLUGIN_NAME>
  For example:

  $ ionic cordova plugin remove cordova-admob-plus 
  $ ionic cordova plugin remove cordova-plugin-admob-free
  $ ionic cordova plugin remove cordova-admob-sdk
